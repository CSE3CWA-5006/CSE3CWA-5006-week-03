# PostgreSQL Prisma CRUD Lab

This lab uses Prisma ORM with PostgreSQL.

Run on Ubuntu:

```bash
chmod +x setup_ubuntu_postgresql_prisma.sh
./setup_ubuntu_postgresql_prisma.sh
npm start
```

The program demonstrates:

- `prisma.student.create()`
- `prisma.student.findMany()`
- `prisma.student.update()`
- `prisma.student.delete()`

Database:

- `university_db`
- `university_user`
- `123456`

The password is for a local classroom VM only.
